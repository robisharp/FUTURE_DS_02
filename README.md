# Customer Retention & Churn Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing customer churn behavior and retention trends for a subscription-based business using Power BI. The dashboard provides actionable insights into customer loss, retention drivers, and customer lifetime value (CLTV).

The objective of this project is to help businesses understand:

* Why customers churn
* Which customer groups are at higher risk
* How retention strategies can improve customer lifetime value

---

## 📊 Dashboard Features

### 🔹 KPI Metrics

* Total Customers
* Churned Customers
* Churn Rate %
* Average Tenure
* Average Monthly Charges
* Customer Lifetime Value (CLTV)

### 🔹 Interactive Visuals

* Churn by Contract Type
* Churn by Payment Method
* Customer Retention Trends
* Monthly Charges vs Tenure Scatter Plot
* CLTV by Contract
* Churn Trend by Tenure Groups

### 🔹 Filters / Slicers

* Gender
* Contract Type
* Internet Service

## 📂 Dataset

---

## 🔍 Key Insights

* Month-to-month contract customers showed the highest churn rate
* Customers with lower tenure were more likely to churn
* Long-term contracts had higher retention and CLTV
* Electronic check payment users showed higher churn behavior

---

## 💡 Business Recommendations

* Encourage customers to move to long-term contracts
* Improve onboarding experience for new customers
* Target high-risk customers with personalized offers
* Promote automatic payment methods to improve retention
